    donna.tasks.db_models¶

    donna.tasks.db_models ¶

    SQLAlchemy models for Donna's task database.

    All tables include user_id from day one for future multi-user support. Schema changes require an Alembic migration — never modify tables manually. See docs/task-system.md and docs/model-layer.md.

    Base ¶

    Bases: DeclarativeBase
